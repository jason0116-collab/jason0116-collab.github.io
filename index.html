AI Insight 대시보드 스냅샷 + 뉴스(Google News) + 분석 프로토콜 → Gemini 종합 판단

「AI 분석 실행」을 누르면 현재 대시보드 상태를 자동 스냅샷하고, 최신 뉴스(기본: Google News · 키 불필요)를 수집한 뒤, 분석 프로토콜과 함께 Gemini에 전달해 종합 판단을 생성합니다. (Gemini API Key는 우측 상단에 입력)
분석 프로토콜 · 뉴스 검색어 · Gemini 모델 설정 (Gemini에 전달되는 지시문 — 편집 시 저장)
프로토콜·검색어·소스·모델은 이 브라우저(localStorage)에만 저장됩니다. Gemini에는 [분석 프로토콜 + 대시보드 스냅샷 + 최근 뉴스]가 하나의 프롬프트로 전달됩니다. 뉴스는 기본 Google News(키 불필요)로 수집되고, 분석에는 Gemini Key만 있으면 됩니다 (우측 상단 입력). GNews를 쓰려면 소스를 바꾸고 키를 입력하세요.
